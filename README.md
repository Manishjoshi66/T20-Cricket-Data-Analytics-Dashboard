# 📊 Dataset Summary

### One of my projects is a T20 Cricket Data Analytics Dashboard where I analyzed player performance data to identify the best possible playing XI for a T20 team.

In this project, I worked with a cricket dataset containing player statistics such as batting average, strike rate, boundary percentage, and average balls faced.

First, I used Python libraries like Pandas and NumPy to clean and preprocess the data. This included handling missing values, structuring the dataset, and preparing it for analysis.

After data preprocessing, I imported the dataset into Power BI and built interactive dashboards to visualize the insights. The dashboard categorizes players into different roles such as Openers, Anchors, Finishers, All-rounders, and Bowlers.

I also implemented interactive features like hover tooltips and cross-filtering, which allow users to explore player statistics dynamically.
